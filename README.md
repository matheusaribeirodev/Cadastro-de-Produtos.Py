# Cadastro-de-Produtos.Py
📦 Sistema de Cadastro de Produtos

Um sistema simples desenvolvido em Python para realizar o cadastro e listagem de produtos utilizando arquivos .txt para armazenamento de dados. O projeto foi criado com foco em praticar lógica de programação, manipulação de arquivos e menus interativos.

🚀 Funcionalidades

Cadastro de produtos
Armazenamento em arquivo .txt
Listagem de produtos cadastrados
Menu interativo no terminal
Sistema simples e intuitivo
